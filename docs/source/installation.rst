.. _installation:

How to install
================

There are two ways to install Fedot.Industrial:

from PyPI
---------
.. code-block:: console

    pip install fedot_ind


from GitHub
------------------------

.. code-block:: console

    git clone https://github.com/ITMO-NSS-team/Fedot.Industrial.git
    cd FEDOT.Industrial
    pip install -r requirements.txt
    pytest -s test/
